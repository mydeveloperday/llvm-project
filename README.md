# The LLVM Compiler Infrastructure

This fork of the LLVM project is to maintain experimental changes for
clang-format and other tools

There is some reluctance to accept all revisions into clang-format, citing that
the clang-format shouldn't support every revisions or that some revision impact
development costs

However there are revisions which seem desiredi by the wider community, but those revisions seem blocked.

The purpose here is to give those revisions some velocity, by the introduction
of clang-format-experimental

Inclusion into the repository, requires revisions submitted to phabricator and
those revisions have not been upstreamed.
